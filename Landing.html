Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get to the equator. To accomplish this analysis, we first pulled the data from the OpenWeatherMap API to assemble a dataset on over 500 cities.



After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at include: temperature, cloudiness, wind Speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and decriptions of any trends and correlations witnessed.
